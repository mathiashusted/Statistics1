Aufgabe 7

A) Sei n der Umfang einer Stichprobe von Studierenden und k die Anzahl der Studentinnen in dieser Stichprobe. Geben Sie fr beliebiges n 4 eine Formel fr den p-Wert des zweiseitigen Binomialtests (H0 : p = 1/2) an, wenn sich in der Stichprobe (a) k = 0 (b) k = 1 (c) k = 2 Studentinnen befinden.

Allgemeine Formel zur Bestimmung von Wahrscheinlichkeit: \(P_r(X=k) = \binom{n}{k}\cdot p^k\cdot(1-p)^{n-k}\\\)

Beispiel fr k = 0: \(P_r(X=0) = \binom{n}{0}\cdot(\frac{1}{2})^0\cdot(1-\frac{1}{2})^{n-0}\)

Also kann die Formel allgemein wie folgt ausgedrckt werden:

\[ \sum_{i=0}^{k}{\binom{n}{k}\cdot p^k\cdot(1-p)^{n-k}} + \sum_{i=k}^{n}{\binom{n}{k}\cdot p^k\cdot(1-p)^{n-k}} \]

Diese Formel wurde an das folgende R Beispiel aus der Vorlesung angelehnt:

p.Wert <- sum(dbinom(0:465, size=1003, prob=0.5))+
+ sum(dbinom(538:1003, size=1003, prob=0.5))

B: Bestimmen Sie (durch systematisches Probieren mit R) das kleinste n, fr das der Binomialtest die Nullhypothese H0 : p = 1/2 auf dem Signi- fikanzniveau = 0,001 ablehnt, wenn (a) k = 0 (b) k = 1 (c) k = 2 ist. Benutzen Sie die Formeln aus der vorigen Teilaufgabe.

Wir knnen mit Hilfe einer Schleife systematisch das kleinste n herausfinden. Probehalber wurde das n hier bis 10.000 getestet, was sich fr k={0,1,2} als ausreichend herausgestellt hat.

nTest <- function(k, sigNiveau) {
  print(paste0("Test fr k = ", k))
  for (x in k:10000) {
    p.Wert <- sum(dbinom(k, size=x, prob=0.5))+
    + sum(dbinom(k, size=x, prob=0.5))
    if (p.Wert < sigNiveau) {
      print(paste0("p-Wert: ", p.Wert))
      print(paste0("Kleinstes n gefunden. n = ", x))
      break
    }
    else {
      print(paste0("p-Wert: ", p.Wert))
      print(paste0(x))
    }
  } 
}
